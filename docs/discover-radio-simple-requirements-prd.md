# Discover Radio 极简需求 PRD

## 电台

| 需求 | 说明 | 优先级 |
|---|---|---|
| 连接 Apple Music | 用户可授权 Apple Music，用于搜索、读取可用歌曲和播放。 | P0 |
| 电台创建入口 | 用户可进入创建电台流程。 | P0 |
| 歌曲 / 播放列表选择 | 用户可选择歌曲或从播放列表中选择歌曲作为电台内容。 | P0 |
| 电台草稿预览 | 发布前展示电台标题、歌曲、讲解、可见性和播放效果。 | P0 |
| 电台播放控制 | 支持播放、暂停和播放状态展示。 | P0 |
| 上一首 / 下一首 | 用户可在电台队列中切换上一首或下一首。 | P0 |
| Apple Music 播放 | 授权和订阅可用时播放 Apple Music 曲目。 | P0 |
| 本地 preview 兜底播放 | Apple Music 不可用时使用本地 preview 或 mock 曲目播放。 | P0 |
| 随机模式 | 电台 agent 读取用户音乐档案中的信息，自行决定下一首歌以及歌单。 | P0 |
| 筛选模式 | 用户可按照歌手、流派、年代等维度筛选歌曲，生成专属电台。 | P0 |

## 发现页

| 需求 | 说明 | 优先级 |
|---|---|---|
| 发现队列加载 | 发现页可加载公开电台队列。 | P0 |
| 左右滑动切换电台 | 用户可左右滑动切换上一个或下一个电台。 | P0 |
| 展示当前公开电台 | 页面展示一个当前正在收听的公开电台。 | P0 |
| 播放当前电台 | 用户可直接播放当前发现页电台。 | P0 |
| 空态 / 失败兜底 | 无网络、无电台或加载失败时展示可播放兜底内容。 | P0 |
| 复古电台 UI | 发现页以老式电台视觉呈现当前频道。 | P0 |
| 分享按钮 | 用户可从发现页分享当前电台。 | P1 |
| 电台详情入口 | 用户可查看当前电台的歌曲和说明。 | P1 |
| 发布者入口 | 用户可进入当前电台发布者主页。 | P1 |
| 收藏电台 | 用户可收藏当前电台。 | P1 |

## 音乐档案

| 需求 | 说明 | 优先级 |
|---|---|---|
| 设置页面 | 用户可管理 Apple Music、数据收集和档案设置。 | P0 |
| 声音档案主页 | 用户可查看自己的音乐档案入口页。 | P0 |
| Apple Music 授权状态 | 展示 Apple Music 授权和订阅状态。 | P0 |
| 音乐数据收集开关 | 用户可开启或关闭音乐数据收集。 | P0 |
| 档案摘要 | 展示用户音乐偏好的简短摘要。 | P0 |
| 常听歌手 / 曲风 / mood 统计 | 展示常听歌手、曲风和 mood 等基础统计。 | P0 |
| 阶段偏好 / 记忆锚点 | 展示阶段性偏好和代表歌曲。 | P1 |
| 已发布电台集合 | 展示用户已发布的电台。 | P1 |
| 收藏电台集合 | 展示用户收藏过的电台。 | P1 |
